前言

昨天作死开始了从wp到Hexo的迁移之旅，网上大部分Hexo搭建教程都还停留在2017年甚至2016年，所以整个迁移过程历尽深坑不断。

因此我觉得有必要记录一下自己所踩过的那些深坑。

具体有以下几点：

  • 搬瓦工ssh端口；
  • ssh公钥和秘钥问题；
  • VPS文件/文件夹权限问题；
  • post-receive钩子无法正常执行的问题；

主要参考：

Hexo环境搭建个人博客

使用 Git Hook 自动部署 Hexo 到个人 VPS
